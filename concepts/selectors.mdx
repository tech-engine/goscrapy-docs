---
title: "Parse HTML with CSS and XPath selectors"
description: "Extract data from HTML responses in GoScrapy using chainable CSS and XPath selectors, attribute lookups, text extraction, and nested queries."
---

The `IResponseReader` embeds `ISelector`, providing chainable methods for querying HTML. Selectors return an `ISelector` which can be further chained or have its data extracted using `.Text()`, `.Attr()`, `.Get()`, or `.GetAll()`.

## Selector API

| Method | Returns | Description |
| :-- | :-- | :-- |
| `Css(selector)` | `ISelector` | Query using a CSS selector. Chainable. |
| `Xpath(expr)` | `ISelector` | Query using an XPath expression. Chainable. |
| `Text(defaults...)` | `[]string` | Extract inner text from matched nodes. |
| `Attr(name)` | `[]string` | Extract an attribute value from matched nodes. |
| `Get()` | `*html.Node` | Get the first matched raw `html.Node`. |
| `GetAll()` | `[]*html.Node` | Get all matched raw `html.Node`s. |

## CSS Selector Examples

### Extracting Text

```go
// Single element
titles := resp.Css(".product_main h1").Text()
if len(titles) > 0 {
    title := titles[0]
}

// Multiple elements
prices := resp.Css("article.product_pod .price_color").Text()
```

### Extracting Attributes

```go
// Get href from links
for _, productUrl := range resp.Css("article.product_pod h3 a").Attr("href") {
    s.Parse(s.Request(ctx).Url(productUrl), s.parseProduct)
}

// Get title attribute
titles := resp.Css("article.product_pod h3 a").Attr("title")
```

### Scoping Selectors (Chaining)

You can scope your selectors by chaining `.Css()` calls, narrowing the search context:

```go
// First, scope to the product page container
product := resp.Css("article.product_page")

// Then extract within that scope
title := product.Css(".product_main h1").Text()
price := product.Css(".price_color").Text()
rating := product.Css(".star-rating").Attr("class")
```

### Advanced CSS Targeting

```go
// Nth-child targeting for table data
upc := resp.Css("table tr:nth-child(1) td").Text()
productType := resp.Css("table tr:nth-child(2) td").Text()
reviewCount := resp.Css("table tr:nth-child(7) td").Text()

// Adjacent sibling selector
description := resp.Css("#product_description + *").Text()
```

### Safe Extraction Pattern

Since `.Text()` and `.Attr()` return slices, always check length before accessing:

```go
var stock string
if stocks := resp.Css(".availability").Text(); len(stocks) > 0 {
    stock = strings.TrimSpace(stocks[0])
}
```

## XPath Examples

```go
price := resp.Xpath("//p[@class='price_color']").Text()
inStock := resp.Xpath("//p[@class='instock availability']").Text()
```

> \[!TIP\] For complex or repetitive extractions, consider using [Declarative Mapping (Gosm)](/concepts/declarative-mapping) instead of writing imperative selector code.
