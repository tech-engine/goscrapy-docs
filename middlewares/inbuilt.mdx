---
title: "Inbuilt Middlewares"
description: "Using GoScrapy's built-in middlewares"
---

GoScrapy comes with several robust middlewares out of the box. You can configure them in `settings.go`.

## Retry Middleware

Automatically retries failed requests or specific status codes with exponential back-off.

```go settings.go
retryMiddleware := middlewares.Retry(middlewares.RetryOpts{
    MaxRetries: 3,
    Codes:      []uint16{500, 502, 503, 504, 522, 524, 408, 429},
    BaseDelay:  1 * time.Second,
})
```

## Dupefilter

Filters out duplicate requests based on the URL and HTTP method.

```go settings.go
dupefilterMiddleware := middlewares.DupeFilter()
```

## MultiCookieJar

Maintains separate cookie sessions. It automatically uses the `CookieJarKey` set on the request.

```go settings.go
cookieMiddleware := middlewares.MultiCookieJar
```
