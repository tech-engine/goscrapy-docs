---
title: "Quickstart: install GoScrapy and run your first spider"
description: "Install the GoScrapy CLI, scaffold a new project, write a spider, and run your first Go web scraper in a few minutes with this quickstart guide."
---

GoScrapy requires **Go 1.26** or higher.

## Get started

<Steps>
  <Step title="Install GoScrapy CLI">
    Install both `goscrapy` and the shorter `gos` alias tools.

    ```bash
    go install github.com/tech-engine/goscrapy/cmd/...@latest
    ```

    Verify installation:

    ```bash
    gos -v
    ```
  </Step>
  <Step title="Create a New Project">
    Generate all necessary files for a new scraper project.

    ```bash
    goscrapy startproject my_scraper
    ```

    When prompted, accept to resolve dependencies (`go mod tidy`).
  </Step>
  <Step title="Write Your Parsing Logic">
    Open `my_scraper/spider.go` and add a `StartRequests` method to seed the initial URL:

    ```go
    func (s *Spider) StartRequests(ctx context.Context) {
        req := s.Request(ctx).Url("https://books.toscrape.com")
        s.Parse(req, s.parse)
    }
    
    func (s *Spider) parse(ctx context.Context, resp core.IResponseReader) {
        s.Logger().Infof("Status: %d, URL: %s", resp.StatusCode(), resp.Request().URL)
        // your extraction logic here...
    }
    ```
  </Step>
  <Step title="Run Your Spider">
    From the project root directory:

    ```bash
    go run main.go
    ```

    You should see GoScrapy's banner followed by your spider's log output. Press `Ctrl+C` to stop.
  </Step>
</Steps>
