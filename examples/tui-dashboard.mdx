This example integrates GoScrapy's telemetry system with a Bubble Tea-based terminal dashboard, providing real-time scraping metrics.

**Key concepts demonstrated:**

- Setting up the `TelemetryHub` with a `StatsCollector`
- Wiring the `tui.Dashboard` as a telemetry observer
- Starting the engine with `gos.StartWithTUI()` instead of `app.Start()`

## main.go

```go
func main() {
    ctx, cancel := context.WithCancel(context.Background())
    defer cancel()

    // Pass `true` to enable the TUI dashboard
    spider, err := with_tui_stats.New(ctx, true)
    // ... error handling

    if err := spider.Wait(true); err != nil && !errors.Is(err, context.Canceled) {
        fmt.Printf("❌ Engine finished with error: %v\n", err)
    }
}
```

## base.go (TUI Integration)

This is the key file. The TUI is wired in `New()` by creating a `TelemetryHub`, adding a `StatsCollector`, and attaching the `tui.Dashboard` as an observer:

```go
func New(ctx context.Context, tuiEnabled bool) (*Spider, error) {
    app, err := gos.New[*Record]()
    // ...

    app.WithMiddlewares(MIDDLEWARES...).
        WithPipelines(PIPELINES...)

    spider := &Spider{ICoreSpider: app, baseUrl: "https://books.toscrape.com"}
    app.RegisterSpider(spider)

    if tuiEnabled {
        // 1. Create a TelemetryHub
        hub := ts.NewTelemetryHub(nil)

        // 2. Register the Stats middleware as a collector
        hub.AddCollector(HttpStats)

        // 3. Print final stats when engine stops
        app.OnEngineStopped(func(ctx context.Context) {
            HttpStats.Print()
        })

        // 4. Create the TUI dashboard and wire it as an observer
        dashboard := tui.New(app.Logger())
        hub.AddObserver(dashboard)
        app.WithTelemetry(hub, nil)

        // 5. Start with TUI instead of regular app.Start()
        go func() {
            _ = gos.StartWithTUI(ctx, app, dashboard)
        }()
    } else {
        go func() { _ = app.Start(ctx) }()
    }

    return spider, nil
}
```

## settings.go

```go
// The Stats middleware collects request/response metrics
var HttpStats = middlewares.NewStatsCollector()

var MIDDLEWARES = []middlewaremanager.Middleware{
    middlewares.Stats(HttpStats), // must be first to capture all traffic
    middlewares.Retry(),
    middlewares.MultiCookieJar,
    middlewares.DupeFilter(),
}
```
