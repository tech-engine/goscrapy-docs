This example demonstrates how to use the `AzureTLS` middleware to spoof TLS fingerprints and bypass advanced bot protection (Cloudflare, Akamai, etc.).

**Key concepts demonstrated:**

- Configuring the Azure TLS middleware for browser fingerprint emulation
- Per-request TLS and proxy override via context

## main.go

```go
func main() {
    ctx, cancel := context.WithCancel(context.Background())
    defer cancel()

    spider, err := fingerprint_spoofing.New(ctx)
    // ... error handling

    if err := spider.Wait(true); err != nil && !errors.Is(err, context.Canceled) {
        fmt.Fprintf(os.Stderr, "❌ Engine finished with error: %v\n", err)
    }
}
```

## settings.go

```go
var azureTLSOpts = &middlewares.AzureTLSOptions{
    Browser: "firefox",
}

var MIDDLEWARES = []middlewaremanager.Middleware{
    middlewares.Retry(),
    // Disable MultiCookieJar when using AzureTLS (it handles cookies internally)
    middlewares.AzureTLS(azureTLSOpts),
    middlewares.DupeFilter(),
}
```

## spider.go (Per-Request Override)

```go
func (s *Spider) Open(ctx context.Context) {
    // Override TLS options per-request via context
    ctx = middlewares.WithAzureTLSOptions(ctx, &middlewares.AzureTLSOptions{
        Browser: middlewares.BrowserFirefox,
        Proxy:   "http://user:pass@myproxy.com:8080",
    })
    req := s.Request(ctx).Url("https://tls.peet.ws/api/all")
    s.Parse(req, s.parse)
}

func (s *Spider) parse(ctx context.Context, resp core.IResponseReader) {
    fmt.Printf("status: %d\n", resp.StatusCode())
    fmt.Println(string(resp.Bytes()))
}
```
