This example (`scrapejsp`) demonstrates how to scrape a JSON API endpoint instead of HTML pages.

**Key concepts demonstrated:**

- Working with JSON API responses using `resp.Bytes()` and `json.Unmarshal`
- Lifecycle hooks (`Open` / `Close`) for automatic spider startup

## main.go

```go
func main() {
    ctx, cancel := context.WithCancel(context.Background())
    defer cancel()

    spider, err := scrapejsp.New(ctx)
    // ... error handling

    // No StartRequests needed — spider.Open() seeds requests automatically
    if err := spider.Wait(true); err != nil && !errors.Is(err, context.Canceled) {
        fmt.Fprintf(os.Stderr, "❌ Engine finished with error: %v\n", err)
    }
}
```

## spider.go

```go
func (s *Spider) Open(ctx context.Context) {
    req := s.Request(ctx).Url("https://jsonplaceholder.typicode.com/todos/1")
    s.Parse(req, s.parse)
}

func (s *Spider) Close(ctx context.Context) {}

func (s *Spider) parse(ctx context.Context, resp core.IResponseReader) {
    fmt.Printf("status: %d", resp.StatusCode())

    var data Record
    err := json.Unmarshal(resp.Bytes(), &data)
    if err != nil {
        log.Fatalln(err)
    }

    s.Yield(&data)
}
```

## Running

```bash
cd _examples/scrapejsp
```
