This example (`scrapejsp`) demonstrates scraping a JSON API endpoint, parsing the response with `json.Unmarshal`, and yielding the result to pipelines.

**Key concepts demonstrated:**

- Working with JSON API responses instead of HTML
- Using `resp.Bytes()` with `json.Unmarshal`
- Lifecycle hooks (`Open`, `Close`)

## Spider Logic

```go
func (s *Spider) Open(ctx context.Context) {
    req := s.Request(ctx).Url("https://jsonplaceholder.typicode.com/todos/1")
    s.Parse(req, s.parse)
}

func (s *Spider) parse(ctx context.Context, resp core.IResponseReader) {
    fmt.Printf("status: %d", resp.StatusCode())

    var data Record
    err := json.Unmarshal(resp.Bytes(), &data)
    if err != nil {
        log.Fatalln(err)
    }

    s.Yield(&data)
}
```
